# Kore
A Unity common library, framework.

## Features
- Kore
  - LayerMaskExtension
- Kore.Conditions
  - LayerCheck
- Kore.Events
  - GameEvents & GameEventListeners
  - Monobehaviour Events
    - Almost all MonoBehaviour message events
    - MonoBehaviourEventTrigger
  - UnityEventRaiser
  - ConditionalEvent
- Kore.Utils
  - GameObjectUtils
    - Activator
    - Destroyer
  - GizmoDrawers
  - Screenshoter
  - SimpleKeyboardEvent
  - SimpleLogger
  - UriOpener
- Kore.Variables
  - ValueAsset
    - ValueCheck
    - ValueAssetTextBinding
  - ReferenceAssets
  - Threshold
